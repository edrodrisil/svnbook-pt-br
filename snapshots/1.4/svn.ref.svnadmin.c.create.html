Nome

svnadmin create — Cria um novo repositório vazio.

Sinopse

svnadmin create REPOS_PATH

Descrição

Cria um novo repositório vazio no caminho dado. Se o diretório informado não existir, ele será criado para você.[60] Desde o Subversion 1.2, o svnadmin cria novos repositórios baseados no sistema de arquivos fsfs por padrão.

Opções

--bdb-txn-nosync
--bdb-log-keep
--config-dir DIR
--fs-type TYPE

Exemplos

Criar um novo repositório é bem fácil:

$ svnadmin create /usr/local/svn/repos

No Subversion 1.0, sempre é criado um repositório do tipo Berkeley DB. No Subversion 1.1, Berkeley DB é o tipo de repositório padrão, mas um repositório FSFS pode ser criado com a opção --fs-type:

$ svnadmin create /usr/local/svn/repos --fs-type fsfs

[60] Lembre-se, o svnadmin só trabalha com caminhos locais e não com URLs.
